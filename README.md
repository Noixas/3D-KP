# 3D-KP
3D Knapsack project by students of Maastricht University 

Update: April 2020
kproblem is a maven project with updated Javafx 11.

## Requirements

- Java version: 11.0.6
- Apache Maven 3.6.3 

## Run:
#### Windows console:

1. cd /repositoryfolder/kproblem
2. mvn clean javafx:run
